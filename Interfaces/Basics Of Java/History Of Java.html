History Of Java
  • Let's see some points that describe the history of java:
    1. James Gosling , Mike Sheridan , and Patrick Naughton initiated the Java language project in June 1991.
    2. Originally designed for small, embedded systems in electronic appliances like set-top boxes.
    3. Initially called Oak and was developed as a part of the Green project
    4. In 1995, Oak was renamed as "Java". Java is just a name not an acronym. Originally developed by James Gosling at Sun Microsystems (which is now a subsidiary of Oracle Corporation) and released in 1995.
    5. JDK 1.0 released in January 23, 1996.
  • What paved the way for Java?

    The trouble with the reigning programming languages such as C and C++ at that time was that they were designed to compiled for specific target i.e. platform. Although it is possible to compile a C++ program for just about any type of platform, to do so a full-fledged C++ compiler targeted for that platform is required. But compilers are expensive and time consuming to create. An easier and cost effective solution was required. In an attempt to find such a solution, James Gosling and others began working on a portable , platform-independent language that could be used to produce code that would run on a variety of processors under different environments. This effort ultimately led to the development of Java.
