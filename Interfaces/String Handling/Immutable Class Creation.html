Immutable Class Creation
  • An immutable class is a final class having final data members and getter methods. 
      //Example of immutable class
												
  public final class Employee{
		final String pancardNumber; // final data member
		
		public Employee(String pancardNumber){
			this.pancardNumber=pancardNumber;
		}

		public String getPancardNumber(){ // getter method
			return pancardNumber;
		}
  }

    The above class is immutable because:
    1. The instance variable of the class is final i.e. we cannot change the value of it after creating an object.
    2. The class is final so we cannot create subclass.
    3. There are no setter methods i.e. we have no option to change the value of the instance variable.
