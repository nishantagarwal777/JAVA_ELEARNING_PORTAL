Class
  • The class is the fundamental unit of Java programs.
  • It is a blueprint or a template for creating different objects which defines its properties and behaviors.
  • A class in java will contain the followings:
    1. Data member
    2. Method
    3. Constructor
    4. Block
  • Representation of a Class : 
      //Syntax of Class

  class Class_name{
	  data_member; 
	  method; 
 	  }

    An example is given below: 
       public class Cube{
	 int length;
	 int breadth;
	 int height;
	 public int getVolume(){	
	 return (length*breadth*height);
	  }

  • The different types of variables available in java are:
    1. Primitive Type: such as int, float, boolean, etc.
    2. Reference Type: such as strings, arrays, or objects

  • A class can contain any of the following variable types.
    1. Local variables: Variables defined inside methods, constructors or blocks are called local variables.
    2. Instance variables: Instance variables are variables within a class but outside any method. These variables are instantiated when the class is loaded.
    3. Class variables: Class variables are variables declared with in a class, outside any method, with the static keyword.
  • Naming Conventions
    1. Variable names are case-sensitive.
    2. Subsequent characters in a variable may be letters, digits, dollar signs, or underscore characters.
    3. The first letter of a class name should be capitalized.
