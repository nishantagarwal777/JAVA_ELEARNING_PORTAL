Constructors
  • A constructor initializes an object when it is created.
  • It has the same name as its class and is syntactically similar to a method.
  • However, constructors have no explicit return type.
  • Types of constructors:

    There are two types of constuctor:
    1. Default constructor.
    2. Parameterized constructor.
  • Default constructor :

    It has no parameters.

  • Parameterized Constructor:

    1. Most often you will need a constructor that accepts one or more parameters.
    2. Parameters are added to a constructor in the same way that they are added to a method.

    Example: 
      //The following example explains constructor taking parameters :
										
   class MyClass {
	  int x;
  		   MyClass(int i ) {
           x = i;
      }
   }       

     public class Demo{
  
	  public static void main(String args[]){
	  
	  	MyClass t1 = new MyClass( 10 );
                MyClass t2 = new MyClass( 20 );
        System.out.println (t1.x + " " + t2.x);
       }
  }

  • Difference between constructor and method:

    CONSTRUCTOR METHOD
    It is used to initialize the state of an object. It is used to expose behavior of an object.
    It does not have a return type. It must have a return type, even if that is void.
    It is invoked implicitly. It is invoked explicitly.
    The java compiler provides a default constructor if you don’t have any constructor. Method is not provided by the compiler in any case.
    The name must be same as the class name. It may or may not be same as the class name.
