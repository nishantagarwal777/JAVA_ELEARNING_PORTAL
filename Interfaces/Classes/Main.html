Public Static Void Main
  • Meaning of each tokens:
    1. Class: It is used to declare a class in java
    2. Public: It is an access modifier which represents visibility, it means it is visible to all
    3. Static: It is a keyword.If we declare any method as static,it is known as static method
    4. Void : It is the return type of the method,it means it doesn’t return any value
    5. Main : It represents the start up program
    6. String agrs[] It is used for command line argument.
    7. System.out.println() : It is used as print statement.out is a static member of the System class. It is a reference variable of the PrintStream class.println() which is a method of the PrintStream class thus can be invoked using the out reference variable.out in turn can can invoked using the name of the System class since it is static in nature.
  • Executing a java program :

    For executing any java program,you need to :
    1. Create the java program.
    2. Install the jdk.
    3. Set the path of the bin directory under jdk.
    4. Compile and execute the program
    •   //Creating hello java program.

  public class Simple{
	  
	  public static void main(String args[]){
		
	  System.out.println (“Hello Java”);
      }
      
  }

    Download
    1. Save this file as Simple.java
    2. To compile: javac Simple.java
    3. To execute: java Simple

  • Can we overload main() method ??

    Yes,by method overloading. You can thus have a number of main methods in a class. 
      //The following example explains how main is overloaded. :
										
   class HelloWorld {
	  public static void main(String args[]){
	  System.out.println )("HelloWorld....!!");
	  }
	}
    class GoodbyeWorld {
	  public static void main(String args[]){
	  System.out.println )("Goodbye Cruel World....!!");
	  }
	}
    • Download
    1. Save this code in a single file called hellogoodbye.java in your javahtml directory.
    2. Compile it with the command javac hellogoodbye.java
    3. Then list the contents of the directory.
      4. HelloWorld.class
      GoodbyeWorld.class
    4. Type java GoodbyeWorld and then type java HelloWorld.
      5. These programs run and execute independently of each other although they exist in the same source code file.
