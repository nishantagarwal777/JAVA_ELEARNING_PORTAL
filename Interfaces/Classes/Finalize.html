The finalize( ) Method :
  • finalize() method in java is a special method that will be called just before an object's final destruction by the garbage collector.
  • It can be used to ensure that an object terminates cleanly.
  • To add a finalizer to a class, you simply define the finalize( ) method. 
    •   //Syntax of the finalize() method :
  
  protected void  finalize( ) {
  
	  //    finalization code here;
 	  }

    Here, the keyword protected is a specifier that prevents access to finalize ( ) by code defined outside its class. This means that you cannot know when or even if finalize ( ) will be executed.
  • finalize() method is defined in java.lang.Object class, which means it available to all the classes for sake of overriding.
  • finalize method is called by garbage collection thread before collecting object and if not intended to be called like normal method.
  • finalize gets called only once by GC thread, if object revive itself from finalize() method than finalize() will not be called again.
  • The garbage collector of jvm collects only those objects that are created by new keyword.
