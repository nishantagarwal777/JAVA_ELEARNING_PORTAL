Super
    Super is a reference variable that is used to refer immediate parent class object.

    Invocation of a superclass constructor must be the first line in the subclass constructor.

  • Syntactical Representation for calling a superclass constructor :

      										
	super () // the superclass no-argument constructor is called. 
   
             or
   
	super  (  parameter list )  //the superclass constructor with a 
                                                   //matching parameter list is called.

    • If a subclass constructor invokes a constructor of its superclass, either explicitly or implicitly,there will be a whole chain of constructors called, all the way back to the constructor of Object. It is called constructor chaining

  • Uses of super() keyword :

    1. super is used to refer immediate parent class instance variable.
    2. super() is used to invoke immediate parent class constructor.
    3. super is used to invoke immediate parent class method.

  • super is used to refer immediate parent class instance

    Instance variable of current class is referred by instance by default, but to refer parent class instance variable we use super keyword to distinguish between parent class instance variable and current class instance variable. 

     // Example to explain :  	
		class Vehicle{ 
   		int speed = 50;
     }
   
	class Bike extends Vehicle{
          int speed = 100;
		  
		  void display() {
		  System.out.println(super.speed); // will print speed of vehicle
		  }
		  
	  public static void main ( String agrs[] ){
	      Bike b=new Bike();
	      b.display();

	  }
}

  • super is used to invoke parent class constructor : 

     // Example to explain :  	
	class Vehicle{ 
   		Vehicle() {
		System.out.println ( “ Vehicle is created ”);
        }
		
	}
   
	class Bike extends Vehicle{
         Bike() {
		 super(); // will invoke parent class constructor. 
		 System.out.println ( “ Bike is created ” );
		 }
		  
		public static void main ( String agrs[] ){
		Bike b=new Bike();
	    }
    }

  • super can be use to invoke immediate parent class method :

    It should be used in case subclass contains the same method as parent class as in the example: 

     // Example to explain :  	
	class Person{
   		void message() {
		System.out.println ( “ Welcome”);
        }
		
	}
   
	class Student extends Person{
         void message() {
		System.out.println ( “ Welcome to java ”);
        }
		 void display(){
		 message(); // will invoke current class message() method 
		 super.message(); // will invoke parent class message() method 
		 }
		  
		public static void main ( String agrs[] ){
		Student s = new Student();
		s.display();
	    }
    }
