Definition
  • In java inner classes declared without any name are called as anonymous inner classes.
  • These classes can be defined within a method or even within an argument to a method.
  • No explicit class declaration statement is available for an anonymous class.
  • Anonymous inner class cannot inherit from a class or implement an interface.
